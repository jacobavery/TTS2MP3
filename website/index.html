Works Offline

All macOS system voices available with zero internet. Automatic fallback ensures your work never stops.

Read Anything

7 Audio Formats

Export as MP3, WAV, FLAC, AAC, M4B, OGG Vorbis, or Opus — with 16 quality presets to match your needs.

Smart Text Processing

Auto-expands "$1,200" to natural language, strips PDF artifacts, normalizes audio levels, and more.

Instant Cache

Audiobook Builder

Create M4B audiobooks with chapter markers, cover art, and full metadata — directly from EPUB files.

Drop files into a folder and they convert automatically. A background daemon monitors for new files and processes them with your current voice and settings.

Reliability

Built to never fail.

🔄

Per-Chunk Retry

Each text chunk retries up to 3 times with exponential backoff. If a later chunk fails, partial output is preserved.

🌐

Offline Fallback

Network drops mid-conversion? macOS voices take over automatically. No lost work, no restarts.

💾

Content Cache

SHA-256 keyed cache means identical text + voice + settings = instant result. No redundant API calls.

✂

Smart Chunking

Text splits at paragraph and sentence boundaries — never mid-word. Configurable pauses between paragraphs.
